# LearnEnglishWordsTelegramBot

Телеграм-бот для изучения английских слов. Проект делался как курсовая работа по Kotlin и заодно как тренировка работы с Telegram Bot API.

## Что умеет бот

- Отправлять пользователю английские слова для запоминания
- Помогать повторять и закреплять лексику
- Демонстрировать базовую логику бота на Kotlin

## Технологии

- Kotlin
- Telegram Bot API

## Как запустить

1. Создать бота через `@BotFather` и получить `BOT_TOKEN`.
2. Указать токен в конфиге/переменной окружения (см. код проекта).
3. Собрать и запустить приложение через Gradle или из IDE.

## Статус проекта

Курсовая работа — основной функционал реализован и служит примером использования Kotlin + Telegram.
После был доделан для школьной программы.
